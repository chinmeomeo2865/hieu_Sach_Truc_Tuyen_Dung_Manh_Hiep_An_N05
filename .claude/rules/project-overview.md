# Project Overview

## Dự án
**Hiệu Sách Chin** — website bán sách trực tuyến, thiết kế tối giản editorial, lấy cảm hứng từ "Cup of Couple".

## Trạng thái hiện tại
- Frontend React + Vite + Tailwind đang chạy ở `src/`
- Backend Express + MongoDB đang ở `backend/`
- Prototype tĩnh gốc vẫn còn ở `index.html` (tham khảo design)
